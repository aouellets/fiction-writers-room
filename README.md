# Fiction Writer's Room

**Scenes, voice, and structure — the craft of long-form storytelling.** — built in-house by [Skill&nbsp;Me](https://skillme.dev).

For novelists and storytellers. Write vivid scenes, develop a distinct voice through ghostwriting, craft speeches and op-eds, pitch a book proposal, and shape narrative for any audience.

⭐ **If this is useful, star the repo** — it's how we gauge what to build next.

## Install

- **From the catalog:** [skillme.dev/pack/fiction-writers-room](https://skillme.dev/pack/fiction-writers-room) — install the whole pack into Claude in one step.
- **With the skills CLI:** `npx skills add aouellets/fiction-writers-room`
- **Manually:** copy any `skills/<slug>/SKILL.md` into your Claude skills directory.

## Skills in this pack

- **[Fiction Scene Writer](skills/fiction-scene-writer/SKILL.md)** — Writes vivid fiction scenes with sensory detail, subtext, and character interiority.
- **[Ghostwriter](skills/ghostwriter/SKILL.md)** — Matches the voice and style of any provided text sample, then writes new content in that voice.
- **[Speech Writer](skills/speech-writer/SKILL.md)** — Writes keynote speeches with narrative arc, memorable lines, and precise timing.
- **[Book Proposal](skills/book-proposal/SKILL.md)** — Writes non-fiction book proposals: hook, market, author platform, chapter outline.
- **[Op-Ed Writer](skills/op-ed-writer/SKILL.md)** — Writes opinion pieces with a clear thesis, evidence, anticipated objections, and call to action.
- **[Sermon / Speech Outline](skills/sermon-writer/SKILL.md)** — Structures talks with three-point outline, illustrations, application, and memorable close.
